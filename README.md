#  Python Mini Projects Collection

This repository contains a collection of beginner-friendly Python projects that I have built to practice programming and problem-solving skills.

## 📂 Projects Included

### 1. 📝 To-Do List
- A simple command-line application to manage daily tasks.
- Features:
  - Add tasks
  - View tasks
  - Delete tasks

### 2. 📇 Contact Book
- Store and manage contacts efficiently.
- Features:
  - Add new contacts
  - Search contacts
  - Delete contacts

### 3. 🧮 Calculator
- A basic calculator for arithmetic operations.
- Features:
  - Addition, Subtraction, Multiplication, Division

### 4. 🔐 Password Generator
- Generates strong and secure passwords.
- Features:
  - Customizable length
  - Mix of letters, numbers, and symbols
